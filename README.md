

1. nvm https://github.com/creationix/nvm 
2. Once You have nvm install latest node via
```bash
nvm install node
nvm use node
```
3. Install parcel
```bash
npm install -g parcel-bundler
```
4. To use, go to an exercise folder and run the index.html

```bash
cd display_average
parcel index.html
```
